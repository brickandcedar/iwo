# Valentine's Day Interactive Page

A fun, interactive Valentine's Day page where the "Yes" button grows bigger every time they click "No" â€” with falling hearts, cute GIFs, music, and playful toast messages. Built with pure HTML, CSS, and JavaScript.

**Live demo:** [brickandcedar.github.io/iwo](https://brickandcedar.github.io/iwo)

## How It Works

- **Click "No"** and watch the Yes button grow bigger with each click
- The No button changes text with guilt-trip messages
- The cute character GIF changes through different emotional stages
- After 5 No clicks, the No button runs away from your cursor
- **Click "Yes"** (after the No button runs away) to see confetti and a celebration message
- Toggle music with the ðŸ”Š button in the bottom-right corner

## Setup for GitHub Pages

1. Go to your repository **Settings**
2. Click **Pages** in the left sidebar
3. Under **Source**, select **"Deploy from a branch"**
4. Select **`main`** branch and **`/ (root)`** folder
5. Click **Save**
6. Wait ~2 minutes for deployment

Your site will be live at: `https://yourusername.github.io/iwo`

## Customization

- Edit `index.html` to change the question or initial message
- Edit `yes.html` to customize the celebration page
- Edit `script.js` to modify button messages, GIF sequences, or behavior
- Edit `style.css` to change colors, fonts, or animations
- Add your own music to the `music/` folder
